## Example Chatbot Workflow

```mermaid
flowchart TD
    A[User sends message] --> B[Chatbot receives input]
    B --> C[Preprocess input]
    C --> D[Detect intent]
    D --> E{Intent type}
    E -->|FAQ question| F[Search FAQ dataset]
    E -->|Product recommendation| G[Ask preference questions]
    E -->|Unknown intent| H[Fallback response]
    F --> I[Generate answer]
    G --> J[Recommend suitable product]
    H --> K[Ask clarification]
    I --> L[Return response to user]
    J --> L
    K --> L
```
